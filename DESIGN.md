# Lab 12 Portfolio Design

## Theme

Clean academic portfolio with warm paper background, restrained blue accents, compact cards, and readable typography.

## Principles

- Keep the page simple and compatible with GitHub Pages.
- Use clear navigation links for every required homework section.
- Prefer accessible color contrast, semantic HTML, and responsive layouts.
- Use a profile avatar by default, with an easy path to replace it with `assets/profile.jpg` or `assets/profile.png`.
- Make content easy to edit before submission.
